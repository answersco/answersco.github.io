---
title: "Epidural vs Subdural Hematoma: - which can cross a suture? - which cannot cross a suture? - which is biconvex (lenticular) - which is biconcave (crescentic) - which can cross midline? - which cannot cross midline?"
---
Epidural Hematoma:
- can NOT cross suture
- biconvex
- can cross midline

Subdural Hematoma:
- CAN cross suture
- biconcave
- cannot cross midline
