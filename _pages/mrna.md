---
title: "MRNA"
---
-encodes amino acid sequence
-carries information from DNA to ribosome
-produced in the nucleus
-transported in the ribosome
-a three-nucleotides codon specifies a particular amino acid
-every sequence starts with AUG
~ 500 to 4500 bases long

-RNA transcribed from DNA that contains the code used in translation
-working copy of the code
-messenger RNA
